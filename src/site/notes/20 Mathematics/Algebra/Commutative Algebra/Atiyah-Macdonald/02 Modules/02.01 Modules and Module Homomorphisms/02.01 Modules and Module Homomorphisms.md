# 02.01 Modules and Module Homomorphisms

Fix a commutative [[20 Mathematics/Algebra/Commutative Algebra/Atiyah-Macdonald/01 Rings and Ideals/01 Rings and Ring Homomorphisms/Ring\|Ring]] $A$. An [[20 Mathematics/Algebra/Commutative Algebra/Atiyah-Macdonald/02 Modules/02.01 Modules and Module Homomorphisms/Module\|$A$-module]] is an abelian group equipped with an action of $A$ that distributes over both additions, respects multiplication, and lets $1$ act as the identity. Equivalently, the action is a ring map $A\to\operatorname{End}_{\mathbb Z}(M)$.

## Why this definition is broad

- Every [[20 Mathematics/Algebra/Commutative Algebra/Atiyah-Macdonald/01 Rings and Ideals/02 Ideals and Quotient Rings/Ideal\|Ideal]] of $A$ is an $A$-module.
- A module over a [[20 Mathematics/Algebra/Commutative Algebra/Atiyah-Macdonald/01 Rings and Ideals/03 Zero Divisors Nilpotents and Units/Field\|Field]] is exactly a vector space.
- A $\mathbb Z$-module is exactly an abelian group.
- A $k[t]$-module is a $k$-vector space together with a chosen linear operator: $t$ acts by that operator.

An [[20 Mathematics/Algebra/Commutative Algebra/Atiyah-Macdonald/02 Modules/02.01 Modules and Module Homomorphisms/Module Homomorphism\|$A$-linear map]] preserves addition and scalar multiplication. Such maps form the [[20 Mathematics/Algebra/Commutative Algebra/Atiyah-Macdonald/02 Modules/02.01 Modules and Module Homomorphisms/Hom Module\|Hom Module]] $\operatorname{Hom}_A(M,N)$. Precomposition reverses the direction in the first variable; postcomposition preserves the direction in the second.

## A useful identification

Evaluation at $1$ gives a natural isomorphism

$$\operatorname{Hom}_A(A,M)\cong M.$$

Indeed, an $A$-linear map $f:A\to M$ must satisfy $f(a)=af(1)$, while every choice of $f(1)$ defines such a map. This is the module version of saying that a linear map out of a one-dimensional free object is determined by one vector.

## Pitfall

The scalar ring is part of the data. The same abelian group can carry different module structures, and a map may be linear for one scalar action but not another. See [[20 Mathematics/Algebra/Commutative Algebra/Atiyah-Macdonald/02 Modules/02.08 Restriction and Extension of Scalars/Restriction of Scalars\|Restriction of Scalars]] and [[20 Mathematics/Algebra/Commutative Algebra/Atiyah-Macdonald/02 Modules/02.08 Restriction and Extension of Scalars/Extension of Scalars\|Extension of Scalars]].
