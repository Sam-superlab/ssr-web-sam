# Cyclic Module

An $A$-module $M$ is **cyclic** if one element $x\in M$ generates it:

$$M=Ax=\{ax:a\in A\}.$$

The map $A\to M$, $a\mapsto ax$, is surjective, with kernel $\operatorname{Ann}_A(x)$. Hence the [[20 Mathematics/Algebra/Commutative Algebra/Concepts/Modules/Submodules and Quotients/First Isomorphism Theorem for Modules\|First Isomorphism Theorem for Modules]] gives

$$M\cong A/\operatorname{Ann}_A(x).$$

So cyclic modules are exactly quotient modules of the regular module $A$ by [[20 Mathematics/Algebra/Commutative Algebra/Concepts/Ideals and Quotients/Operations/Ideal\|ideals]].

## Examples

- $A/I$ is generated by $1+I$.
- $\mathbb Z/n\mathbb Z$ is cyclic as a $\mathbb Z$-module.
- $A^2$ is generally not cyclic; over a field it has dimension two.

A cyclic module is the one-generator case of a [[20 Mathematics/Algebra/Commutative Algebra/Concepts/Modules/Finite Generation and Local Methods/Finitely Generated Module\|Finitely Generated Module]].
