#con 
## Definition
A structural model of memory asserts that memory consists of the central executive, visuospatial sketch pad, phonological loop, and [[90 Archive/Adjacent Fields/Psychology 2/Psychology Revision/Concepts/Episodic buffer\|Episodic buffer]].
