## Basic concepts
>**Set**: is a collection of objects, each object is called **elements** of set.

### Some basic notation:
- $\in$ means *an element is in*
- $\not \in$ means *an element is not in*
- $n(S)$ or $|S|$ means the number of elements in set $S$
